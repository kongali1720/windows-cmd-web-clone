## 🚀 Fitur Utama

- **🖼️ Antarmuka Klasik Windows 9x/2000:** Tampilan jendela, bilah judul, dan tombol kontrol klasik.  
- **📂 Menu dan Toolbar Fungsional:** Menu File, Edit, View, dan toolbar interaktif (walau simulasi).  
- **⌨️ Interaksi Terminal:** Ketik perintah dan lihat output yang disimulasikan.  
- **📱 Responsif:** Tampilan menyesuaikan ukuran layar.  

---

## 🛠 Teknologi yang Digunakan

- **HTML5** – Struktur halaman.  
- **CSS3** – Styling retro.  
- **JavaScript** – Interaktivitas, simulasi perintah, kontrol jendela.  

---

## 💻 Cara Menjalankan Proyek

1. **Clone Repositori:**  
```bash
git clone https://github.com/kongali1720/windows-cmd-web-clone.git
```
